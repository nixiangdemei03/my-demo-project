# Project: RBT

## What this is
RBT 是一个综合性项目，旨在为用户提供前后端一体化的解决方案。项目面向开发者团队，解决快速搭建与协作开发的问题。

## Tech stack
- Backend: FastAPI + PostgreSQL
- Frontend: React + Vite
- Deploy: Cloudflare Pages + Workers

## Project structure
- `backend/` - FastAPI app
- `frontend/` - React app
- `模板/` - 模板文件
- `makedown/` - Markdown 文档
- `技术使用说明文件/` - 技术使用说明文档
- `weekly-report/` - 周报文件夹

## Commands
- `make dev` - run all locally
- `make test` - run all tests
- `make deploy` - deploy to Cloudflare

## Conventions
- Use Conventional Commits (feat:, fix:, docs:, ...)
- All endpoints require tests
- All PRs need green CI before merge
