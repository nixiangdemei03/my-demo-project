# Issue 09: Frontend — Homepage, Product Catalog & Detail Pages

**Label**: `phase-1` `P0`

## 验收标准

### 首页 `/`
- [ ] 平台名称 + 一句话定位
- [ ] 三个搜索入口（Tab 切换或三栏）：
  - VIN 搜索框（输入 VIN → 跳转结果页）
  - OEM/文字搜索框（输入关键词 → 跳转结果页）
  - 车型浏览入口（品牌→车型→年份 下拉级联）
- [ ] 热门类别快捷入口

### 搜索结果页 `/search`
- [ ] 列表/网格视图切换
- [ ] 左侧筛选栏：类别树、价格区间、供应商
- [ ] 右侧结果列表：产品卡片（缩略图、名称、OEM、价格、MOQ、供应商）
- [ ] 分页
- [ ] URL 参数同步（`?q=刹车片&make=Toyota`）

### 产品详情页 `/products/:id`
- [ ] 多图轮播（点击放大）
- [ ] 产品信息：名称（中/英）、OEM 编号、描述
- [ ] 价格表：CNY 原价 + USD 售价、MOQ
- [ ] 规格表（从 specs JSONB 渲染键值对）
- [ ] 车型适配列表（品牌/车型/年份/发动机）
- [ ] 供应商信息卡片（Logo、公司名、verified 徽章、已上架产品数）→ 点击跳转供应商页
- [ ] 保修/退换政策
- [ ] "发送询问"按钮 → 跳转询问表单

### 供应商详情页 `/suppliers/:id`
- [ ] Logo、公司名、verified 状态
- [ ] 公司介绍
- [ ] 主营品牌标签
- [ ] 资质证书缩略图
- [ ] 该供应商产品列表

### 登录 `/auth/login` + 注册 `/auth/register`
- [ ] 邮箱 + 密码登录
- [ ] 注册：角色选择 + 公司信息 + 联系人
- [ ] 表单验证（密码强度、邮箱格式）
- [ ] 错误提示（邮箱已注册、密码错误等）

### 响应式
- [ ] PC 端（≥1024px）：侧边筛选栏 + 结果区
- [ ] 移动端（<768px）：顶部搜索 + 底部弹出筛选
