## 验收标准

### 工作台 `/dashboard`
- [ ] 概览卡片：产品总数、待处理订单数、本月销售额
- [ ] 最近订单列表（前 5 条）

### 产品管理 `/dashboard/products`
- [ ] 产品列表（搜索、分页、按状态筛选 active/inactive）
- [ ] 添加产品按钮 → 跳转表单页
- [ ] 添加产品表单 `/dashboard/products/new`
  - 名称（中/英）、类别下拉选择、OEM 编号
  - 原价(CNY)、售价(USD)、MOQ、库存
  - 规格键值对（动态添加行：key + value）
  - 描述（中/英）富文本
  - 车型适配绑定（品牌/车型/年份起止/发动机/VIN pattern，动态添加行）
  - 图片拖拽上传（预览、排序、设封面）
  - 保修期、退换政策
  - 提交后跳转产品列表
- [ ] 编辑产品 `/dashboard/products/:id/edit`
  - 预填充所有字段
  - 可新增/删除图片
  - 保存后跳回列表
- [ ] 下架/删除产品（确认弹窗）

### 订单管理 `/dashboard/orders`
- [ ] 收到的订单列表（状态标签颜色区分）
- [ ] 订单详情 `/dashboard/orders/:id`
  - 采购商信息（公司名、联系人、国家）
  - 产品信息（名称、数量、单价、总价）
  - 询问内容（OEM 号、需求图片、车型信息）
  - 操作按钮：确认订单 / 拒绝（附原因输入）
  - 已确认订单：发货按钮 → 选择货运公司 + 填运单号
  - 订单时间线
