# APEX — 实施路线图

## Phase 1: 基础平台 (Week 1-2)

- [P0] 项目脚手架：FastAPI + React + Vite，跑通 hello-world
- [P0] PostgreSQL + SQLAlchemy + Alembic 配置
- [P0] 用户注册/登录 API（JWT + bcrypt + 邮箱验证）
- [P0] 产品 CRUD API（含图片上传至 Cloudflare R2）
- [P0] 产品分类 API
- [P0] 前端：首页、产品目录、产品详情页
- [P0] 前端：登录/注册页面
- [P0] 前端：供应商产品管理页（添加/编辑/列表）

## Phase 2: 订单与货运 (Week 3-4)

- [P0] 订单 API（创建/查询列表/状态流转）
- [P0] 货运公司 CRUD API
- [P0] 发货与追踪 API（关联货运公司+运单号+追踪事件）
- [P0] 前端：采购商下单流程
- [P0] 前端：订单详情 + 货运追踪展示
- [P1] 询盘 API + 前端消息列表
- [P1] 管理后台：用户审核、订单全局视图

## Phase 3: 完善与部署 (Week 5-6)

- [P1] 管理后台：统计看板、类别管理
- [P1] 邮件通知（注册验证、订单状态更新）
- [P1] 前端英文版（国际化）
- [P0] 前端部署至 Cloudflare Pages
- [P0] 后端部署至 Cloudflare Workers / VPS
- [P1] 集成 1-2 家货运公司 API（自动拉取追踪事件）
